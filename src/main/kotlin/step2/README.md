기능 정리

- 입력 (InputView)
    - 구매 금액을 입력한다.
    - 당첨 번호를 입력한다.
- 출력 (OutputView)
    - 구매금액 / 1000 계산식을 이용한 구매 개수를 출력한다.
    - 로또번호 6자리를 출력한다.
    - 당첨 통계를 출력한다.
- 로또 (Lotto)
    - 1~45 사이의 중복 없이 정렬된 무작위 숫자 6가지를 가지고 있는다.
- 통계 (Statistics)
    - 3개 일치 (5000원)
    - 4개 일치 (50000원)
    - 5개 일치 (1500000원)
    - 6개 일치 (2000000000)
